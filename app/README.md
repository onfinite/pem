# Pem mobile app (`app/`)

Expo + React Native client. **Product story, stack, and env** are documented in the **repo root [`README.md`](../README.md)**.

## Commands (from `app/`)

```bash
npm install
npx expo start
npm run lint
npm test
```

Path alias **`@/`** → this package root (`tsconfig.json` / `babel.config.js`).
